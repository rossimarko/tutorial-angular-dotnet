## üÜò Troubleshooting

### Common Issues:

**Issue**: Docker won't start
- **Solution**: Ensure Docker Desktop is installed and running
- **Module**: 01_environment_setup.md ‚Üí Docker Setup

**Issue**: API won't connect to database
- **Solution**: Check SQL Server container is running and connection string is correct
- **Module**: 04_sql_server_dapper.md ‚Üí Database Setup

**Issue**: Angular ng serve fails
- **Solution**: Clear node_modules and reinstall: `rm -r node_modules && npm install`
- **Module**: 06_angular_setup.md ‚Üí Troubleshooting

**Issue**: JWT token validation fails
- **Solution**: Verify secret key is same in appsettings and auth service
- **Module**: 05_authentication_jwt.md ‚Üí Token Validation
